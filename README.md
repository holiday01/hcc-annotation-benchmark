# HCC scRNA-seq Cell Type Annotation Benchmark

Companion code for the manuscript on benchmarking cell type annotation
tools for hepatocellular carcinoma (HCC) single-cell RNA sequencing,
spanning reference-based classifiers, marker-database scoring, a Bayesian
marker-gene model, a large language model API, and a single-cell
foundation model, evaluated under both author-curated and database-derived
ground truth.

> **Note**: This repository contains analysis code only. Manuscript and
> figures will be released after acceptance.

---

## Repository Structure

```
.
├── scripts/                  # Analysis pipeline (numbered execution order)
├── ground_truth/             # Author-curated cell type labels (re-formatted from source papers)
│   ├── GSE125449_ground_truth.tsv
│   └── GSE149614_ground_truth.tsv
├── external/sc-type/         # Third-party ScType R scripts and marker DB
│                             # (from https://github.com/IanevskiAleksandr/sc-type)
├── requirements.txt          # Python dependencies
├── LICENSE                   # MIT
└── README.md
```

The `enrichR-derived` ground truth files are not shipped — they are
regenerated by `scripts/10_build_enrichr_groundtruth.py` from the raw
data.

## Datasets

All data are publicly available from NCBI Gene Expression Omnibus (GEO).
Download them before running the pipeline:

| Dataset | GEO accession | Cells | Author labels |
|---|---|---:|---|
| GSE125449 | [GSE125449](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE125449) | 9,752 | Yes |
| GSE149614 | [GSE149614](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE149614) | 71,915 | Yes |
| GSE156625 | [GSE156625](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE156625) | 73,589 | No |
| GSE162616 | [GSE162616](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE162616) | 57,261 | No |
| GSE202642 | [GSE202642](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE202642) | 122,092 | No |
| GSE223204 | [GSE223204](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE223204) | 23,165 | No |

## Installation

### Python (3.13 tested)

```bash
pip install -r requirements.txt
```

### R (4.3 tested)

```r
install.packages(c("Seurat", "remotes", "SeuratDisk"))
remotes::install_github("mathewchamberlain/SignacX")
BiocManager::install(c("SingleR", "celldex"))

```

```bash
pip install huggingface_hub
python -c "from huggingface_hub import snapshot_download; snapshot_download('wanglab/scGPT-human', local_dir='models/scGPT-human')"
```

## Pipeline (numbered execution order)

```bash
# 1. Preprocess all benchmark datasets
python scripts/01_preprocess_benchmark_datasets.py

# 2. Build enrichR-derived ground truth (regenerates ground_truth/*_enrichr_*.tsv)
python scripts/10_build_enrichr_groundtruth.py

# 3. Run annotation tools on benchmark datasets
python scripts/06_run_CellTypist_on_benchmark.py
python scripts/17_run_scType_on_benchmark.py
python scripts/05_run_CellAssign_on_benchmark.py
python scripts/05_run_scGPT_annotation.py
python scripts/04_run_CyteType_on_benchmark.py
Rscript scripts/03_run_signacX_on_benchmark.R
Rscript scripts/14_run_SingleR_on_benchmark.R

# 4. Prepare and run on the external GSE156625 cohort
python scripts/16_prepare_gse156625_external.py
python scripts/18_run_CellAssign_GSE156625.py
Rscript scripts/19_run_SignacX_GSE156625.R
python scripts/20_run_CyteType_GSE156625.py

# 5. Independent malignant-cell validation via inferred CNV
python scripts/22_cnv_malignant_validation.py

# 6. Splatter-simulated negative control
Rscript scripts/21_simulate_scrnaseq_splatter.R
python scripts/23_run_tools_on_simulated.py
Rscript scripts/24_run_singler_signacx_on_simulated.R

# 7. Evaluation and figures
python scripts/02_benchmark_evaluation.py
python scripts/09_per_class_analysis.py
python scripts/12_coverage_robustness_analysis.py
python scripts/07_generate_figures.py
python scripts/15_generate_robustness_figures.py
```
